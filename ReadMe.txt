SoapResponseMarshal Project
===========================
The purpose of this project is to illustrate how
to use Camel/Fuse to read SOAP requests from a 
file system, pass those SOAP requests to a web
service, marshal the result to JSON, and 
write the result to the filesystem.

To build this project use

    mvn install

To run this project with Maven use

    mvn camel:run

For more help see the Apache Camel documentation

    http://camel.apache.org/
